Identify User Preferences¶

Here user preference plays a major role as it is required to suggest similar movies to them. Recommending similar movies based on user's interest will improve the system better and will of user's convenience. For example, if a user has rated the movie Titanic with 5, then the most similar movie to Titanic if not watched by the user has a high chance to be watched when recommended to that user.

Creating a function to identify the top preferred (identified by movies with user's ratings with 4 & 5) movies for the users. This function takes input the userid and top N movies rated 4& 5 by the user. This returns the top N most liked movies for that user.

In [845]:
def most_preferred(userid, top_n):
    high_rated = max(ratings.loc[(ratings.userId==userid),'rating'])  #Assign maximum of rating given by the user 
    if high_rated < 4:                                                #If the rating < 4, the movie is not liked by the user
        print('Not likely')
        return(None)
    else:
        movie5 = ratings.loc[(ratings.userId==userid)&(ratings.rating==5),'movieId']  #Movies with rating 5 by the user
        movie4 = ratings.loc[(ratings.userId==userid)&(ratings.rating==4),'movieId']  #Movies with rating 4 by the user
       
        #If the movies rated 5 exceeds top N we return a random top N movies
        if len(movie5) > top_n:                    
            return(list(np.random.choice(movie5,top_n)))
        
        #If the movies rated 4 & 5 together is less than top N, return both lists of movies together
        elif len(movie5) + len(movie4) < top_n:    
            return(list(movie5)+list(movie4))
        
        #Else list all movies rated 5 first then, randomly select movie rated 4 for rest of the top N movies
        else:
            return(list(movie5) + list(np.random.choice(movie4,top_n-len(movie5))))

Recommendations¶

To provide user-user recommendations we define a function using similar movies watched list and similar users list. We have run two loops here:

  • for movies in columns to find their ratings by similar users
  • similar users in index for movies only with rating values from similar_user_movies matrix Calculating the scores by multiplying the similarity scores and ratings given. Then find the sum of the scores and count of the scores given. The weighted average of the user is calculated from sum of scores and count.

The movie ratings are weighted by the similarity scores, so the users with higher similarity get higher weights.

In [573]:
def movie_score(similar_user_movies,similar_users):
    movie_scores = {}                                # Create dictionary to store item scores

    # Loop through movies from user similarity matrix
    for i in similar_user_movies.columns:
        rating = similar_user_movies[i]               # Get the ratings for each movie 
        total = 0                                     # Create a variable to store the score
        count = 0                                     # Create a variable to store the number of scores
     
    # Loop through similar users
      for j in similar_users.index:
        if pd.isna(rating[j]) == False:               # If the movie has a rating
            score = similar_users[j] * rating[j]      # Score is the sum of user similarity score * movie rating
            total += score                            # Add the score to the total score for the movie so far
            count +=1                                 # Add 1 to the count
      
    # Calculate average score for the movie
      movie_scores[i] = total / count
    return movie_scores
